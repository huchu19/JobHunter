# Claude Code instructions

The full operating manual — stack, project map, commands, conventions, and the
milestone-driven workflow — lives in [AGENTS.md](AGENTS.md). Read it before making
changes. Track and update progress in [MILESTONES.md](MILESTONES.md): every feature
change must trace back to a milestone there, and you must run the "Definition of
Done" checklist before marking anything complete.

## Git

When committing and pushing to GitHub, do **not** add Claude (or any AI) as a
co-author. Commits are authored solely by the repository owner — no
`Co-Authored-By: Claude` trailer, no "Generated with Claude Code" line.
